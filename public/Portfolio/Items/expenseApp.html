Expense Report Application - Scinomix

Custom in-house Android and iOS application for Scinomix to log company expenses. Admin web app for designated employees to manage and create expense reports.

Dev Overview

Mobile Application:
Expense Mobile App was created with the Flutter framework. Allows users to login with company email, see past expenses, create new expenses, or save a created expense locally to upload later. User can configure their Google drive folder where uploaded expense receipts are saved.

Admin Web App:
Admin web app allows select employees to login to their account with admin access. These employees can view all expenses, create expense reports, export to Google sheets, view receipts, mark as complete, edit and delete, etc. Gives these admins control over viewing and recording company expenses.
