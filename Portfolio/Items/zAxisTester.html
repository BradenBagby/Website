C++ Z-Axis Tester - Scinomix

C++ App running on Beaglebone to control Motors and test hardware. Allows user to configure test scenerio, start testing, and view progress/status/results on local webpage.

Dev Overview

Written in C++. GUI using QT. Controls serial motors and hardware reading simple IOs. Runs on Beaglebone embedded system with Linux OS. Hosts local web app with Apache and PHP to show current status and test results, allowing user to view logs as well. User can use web app to start or stop test, and configure test case using phsyical touch screen connected to Beaglebone. Created for Scinomix to test hardware for production.
