Flutter - VoipKit Plugin

Flutter Voip Kit is a custom plugin I wrote to use iOS CallKit and Android's Telecom library to create and receive calls with native functionality. e.g. Calls pop up on user's lock screen The Android version was mostly taken from an existing open source plugin, while the iOS version was written in Swift from scratch.
